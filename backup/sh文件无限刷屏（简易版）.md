## 完整代码
```
#!/bin/bash
message="我爱你"
while true; do
    echo $message
    sleep 0.5
done
```

## 代码解析

> ### 1.Shebang (#!) 行
`#!/bin/bash`
这一行告诉操作系统这个脚本应该使用 /bin/bash 解释器来执行。它确保了即使在不同的环境中，脚本也能正确运行。
### 2.定义变量 message
`message="我爱你"`
这里我们定义了一个名为 message 的变量，并给它赋值为字符串 "我爱你"。这是将要在终端中不断显示的信息。
### 3.无限循环 (while true; do ... done)
```
while true; do
    ...
done
```
while true 创建了一个无限循环。只要条件 true 始终成立（即永远），循环体内的命令就会不断地被执行。
do 和 done 是循环结构的一部分，用来界定循环体内包含哪些命令。
### 4.输出信息 (echo $message)
`sleep 0.5`
sleep 命令会让脚本暂停指定的时间（以秒为单位）。这里设置为 0.5 秒，意味着每次打印后会等待半秒钟再进行下一次打印。
这个延迟可以防止信息打印得太快而难以阅读，同时也可以减少系统资源的占用。
### 5.结束循环
当 done 关键字被遇到时，控制权返回到 while 开始处，检查条件是否仍然满足（在这个例子中总是满足），然后再次执行循环体内的命令。
### 6.如何修改刷屏内容
如果你想改变刷屏显示的信息，只需要编辑 message="我爱你" 这一行中的文本。例如：
`message="我爱编程"`
保存文件后，当再次运行脚本时，它将会打印新的信息。
