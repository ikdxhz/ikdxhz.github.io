# Python库管理器 v1.86

![Python版本要求](https://img.shields.io/badge/Python-3.6.1%2B-blue)
![授权协议](https://img.shields.io/badge/License-MIT-green)

作者：ikdxhz  
一个功能强大的Python包管理工具，支持多源切换、批量更新、依赖检查等高级功能

## 功能特性

### 核心功能
- 🔄 多镜像源支持（阿里云/清华/豆瓣/USTC/官方源）
- 📦 包管理全功能：
  - 智能安装（版本指定/包名验证）
  - 安全卸载（依赖关系检查）
  - 单包更新（版本对比）
  - 批量更新（多线程加速）
- 📊 信息查询：
  - 已安装包列表（columns格式）
  - 包详细信息展示
  - 过时包检测

### 增强功能
- 🌐 网络智能检测（自动重试机制）
- 📢 公告系统（远程内容获取）
- 💬 随机一言API集成
- 🛠 自适应pip命令检测（自动尝试pip/pip3）
